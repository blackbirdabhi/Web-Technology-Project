What is a String?

In java, string is basically an object that represents sequence of char values. An array of characters works same as java string.
Example:
char[] ch={'h','e','l','l','o'};
String s=new String(ch);
is same as:
String s="hello";
The java String is immutable i.e. it cannot be changed but a new instance is created.
Generally, string is a sequence of characters. But in java, string is an object that represents a sequence of characters. String class is used to create string object.



Declaration:

There are two ways to create String object:

  • By string literal:

    • Java String literal is created by using double quotes. For Example:
    String s="welcome";



  • By new keyword :

    • String s=new String("Welcome");//creates two objects and one reference variable

    In such case, JVM will create a new string object in normal heap memory and the literal "Welcome" will be placed in the string constant pool. The variable s will refer to the object in heap.





    Basic String Methods:

  • length() returns the length of the string

  • toLowerCase() converts the string to lower case

  • toUpperCase() converts the string to upper case

  • replace(char, char) replaces occurrences of one character with another character.

  • concat(String) method allows two strings to be concatenated together

    • Substrings of a String object can also be accessed

  • A portion of String object can be copied to a character array using the getChars() method

  • The substring() method can return substring beginning at a specified offset

    • Methods of searching strings


  • indexOf(x) searches for the first occurrence of x
  • indexOf(x, y) searches for the first occurrence of x after the offset of y
  • lastIndexOf(x) searches backwards for the first occurrence of x
  • lastIndexOf(x, y) searches backwards for the first occurrence of x after the offset of y
