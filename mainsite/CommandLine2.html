class CommandLine2
{
 public static void main(String args[])
  {
   String s;
   int x=0,i;
   while(x<args.length)
    {
     s=args[x];
     int c=0;
     for(i=0;i<s.length();i++)
      {
       if(s.charAt(i)!=' ')
        {
        c++;
       }
     }
     System.out.println("No. of words of "+args[x]+"="+c);
     x++;
    }
  }
}




EXPLANATION:

In this program, we will initialise a variable which will act as a count to zero.
