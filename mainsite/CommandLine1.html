class CommandLine1
{
 public static void main(String args[])
  {
  int a,b,c;
  a=Integer.parseInt(args[0]);
  b=Integer.parseInt(args[1]);
  c=Math.max(a,b);
  System.out.println("largest no. ="+c);
  }
}




EXPLANATION:

In the above program we have three variables a, b and c. a and b are the inputs given using command line argumnet and c stores the largest of a and b. When the 1st no. is given a will get its value and b gets the value of the 2nd no. The output is printed by System.out.println.
