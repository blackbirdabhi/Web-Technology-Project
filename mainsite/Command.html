What is Command Line Argument?

Suppose we need our program to behave in a certain way depending on some values we pass at the time of execution. This can be achieved by using command line arguments. Early when we created our first program in Java, we run it using the command “java MyFirstJavaProgram”. Here no values are supplied to this program. If we want to pass some values to the Java program, we can supply it as command line arguments. So, it provides a convenient way to check the behavior of the program for the different values. You can pass N (1,2,3 and so on) numbers of arguments from the command prompt. The statement String args[] simply represent a parameter named args which contains an array of String objects. In other words, the main method can take an array of String objects as its input argument.
Suppose we run the program as “java CommandLineArgumentsDemo 1 testing”. So here the values “1” and “testing” are put into a String array and passed to the main method as its argument. These elements can be accessed as “args[0] and “args[1]” respectively. Our program accesses these values using “args[0]” , “args[1]” . It outputs these values using System.out.println command. You can pass any number of command line arguments and these elements can be accessed as “args[0]” , “args[1]” , “args[2]” etc.
