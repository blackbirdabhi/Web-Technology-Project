EXPLANATION:

The java code above utilizes the Euclidean algorithm: that the GCD(a,b)=GCD(a-b,b). It also uses the fact that GCD(a,a)=a and a*b=GCD(a,b)*LCM(a,b). The GCD is found recursively: if a!=b, find GCD of the lower form, until a=b. This is guaranteed by the Euclidean algorithm.
